# 闲鱼商品：Claude Code 每日工作总结系统

## 标题
Claude Code自动生成每日工作总结，配网页+邮件，每天躺着收日报

## 文案

**你是不是也这样？**

每天下班前要写日报，回想今天干了啥，翻聊天记录、翻 git 提交，凑半天才憋出几行字。烦不烦？

这个东西就是帮你解决这个问题的。

**它是什么？**

我花了一整天，用 Claude Code 搭了一套完整的日报自动生成系统。核心就一个效果：

> 你正常写代码，每晚 10 点自动收到一份写好的日报。

具体能干啥：

1. 自动读你的 git 提交记录，理解你今天改了哪些文件、做了什么功能
2. 调用 Claude AI 写日报，不是那种模板填字，是真的理解你的代码后用人话写出来
3. 生成一个网页，青春配色，表格展示，带倒计时，手机电脑都能看
4. 自动发邮件，每晚 10 点日报准时到你邮箱
5. 自动存档，每晚 21:55 自动帮你 commit 代码

**10 块钱包含什么？**

1. 帮你装好整套系统，配好 Claude Code skill
2. 网页界面搭好，你浏览器打开就能看
3. 邮箱通知配置好（QQ/163/Gmail 都行）
4. 教你怎么用，包售后，用不起来退钱

**适合什么人？**

- 每天要写日报的程序员
- 想用 Claude Code 但不知道怎么自定义 skill 的人
- 单纯懒得每天回忆自己干了啥的人
